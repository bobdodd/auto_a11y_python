ErrInputSingleSideBoxShadow - Violations

This page contains text input fields with box-shadow focus indicators on only one side. WCAG Conformance Requirement 5.2.4 requires focus indicators to follow the element shape (all sides). All 5 inputs should be flagged.

Single-Sided Box-Shadow Focus Indicators

Expected Test Results:

  • Error Code: ErrInputSingleSideBoxShadow
  • Count: 5 violations
  • WCAG: 2.4.7 Focus Visible (Level AA), Conformance Requirement 5.2.4
  • Why: Single-sided box-shadows don't follow the element shape. Focus indicators must surround the element on all sides.
  • Horizontal offset: Should be 0 OR vertical offset should be 0, but not both non-zero in same direction
