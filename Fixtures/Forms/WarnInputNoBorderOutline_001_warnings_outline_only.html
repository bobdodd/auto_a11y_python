WarnInputNoBorderOutline - Warnings

This page contains text input fields that use outline-only focus indicators. While these meet WCAG 2.4.7 technically, they're not ideal for screen magnifier users who may only see one field at a time and cannot compare focused vs unfocused states. Border thickening or box-shadow additions are RECOMMENDED. All 4 inputs should generate warnings.

Outline-Only Focus Indicators

Expected Test Results:

  • Warning Code: WarnInputNoBorderOutline
  • Count: 4 warnings
  • WCAG: 2.4.7 Focus Visible (Level AA)
  • Why: Screen magnifier users may only see one input field at a time (zoomed to 400%) and cannot compare focused vs unfocused visual states
  • Recommendation: Use border thickening or box-shadow in addition to or instead of outline
  • Note: This is a usability concern, not a strict WCAG violation
