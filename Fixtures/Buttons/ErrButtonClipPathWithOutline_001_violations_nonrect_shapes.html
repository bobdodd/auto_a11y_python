Test: Clip-Path Buttons with Outline (WCAG 5.2.4 Violation)

These buttons use clip-path to create non-rectangular shapes, but use CSS outline for focus indication. The outline draws a rectangular box that doesn't follow the clipped shape, creating disconnected "ears" or gaps. At high magnification (400%+), users may see only the outline without the button, making it impossible to determine what has focus. This fails WCAG Conformance Requirement 5.2.4.
