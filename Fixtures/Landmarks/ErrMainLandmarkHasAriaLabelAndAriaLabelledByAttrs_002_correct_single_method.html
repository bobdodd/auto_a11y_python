When a page has only one main landmark, it doesn't require aria-label or aria-labelledby.

This is the most common pattern for main content areas.

Best Practice: Main landmarks typically don't need labels when only one exists. If multiple main landmarks are present (uncommon), use aria-label or aria-labelledby, but never both.
