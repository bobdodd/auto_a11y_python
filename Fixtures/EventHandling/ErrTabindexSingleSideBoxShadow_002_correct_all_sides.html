ErrTabindexSingleSideBoxShadow - Correct Examples

This page contains elements with tabindex that use all-sides focus indicators. None should generate errors.

All-Sides Focus Indicators

Correct 1: Uniform Box-Shadow

box-shadow: 0 0 0 3px #0066cc - visible from all directions

Correct 2: Standard Outline

outline: 3px solid #cc0000 - inherently surrounds all sides

Correct 3: Multi-Layer Uniform Shadow

Multiple box-shadows, both uniform (0 0 0 spread)

Correct 4: Uniform Blur Shadow

box-shadow: 0 0 8px 4px - blur applied uniformly to all sides

Correct 5: Inset Uniform Shadow

Inset box-shadow with uniform spread on all sides

Expected Test Results:

  • Error Code: None (negative test)
  • Count: 0 errors
  • WCAG: CR 5.2.4 (Common Requirement for focus indicators)
  • Why: All focus indicators are visible from all sides
  • Details:
    • Correct 1: Uniform box-shadow (0 0 0 3px) - no offset
    • Correct 2: Standard outline - surrounds element
    • Correct 3: Multiple uniform shadows
    • Correct 4: Uniform blur (0 0 for X/Y offsets)
    • Correct 5: Inset uniform shadow
  • Best Practice: Use box-shadow: 0 0 0 Npx or outline for maximum compatibility with zoom/magnification
