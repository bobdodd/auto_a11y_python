ErrTabindexOutlineWidthInsufficient - Correct Examples

This page contains elements with tabindex that have sufficient outline widths (≥ 2px) or use box-shadow. None should generate errors.

Sufficient Outline Width

Correct 1: 2px Outline

outline: 2px solid #0066cc - meets AAA minimum requirement

Correct 2: 3px Outline

outline: 3px solid #cc0000 - exceeds minimum, great visibility

Correct 3: 4px Outline

outline: 4px solid #006600 - excellent visibility for all users

Correct 4: "medium" Keyword

outline: medium solid #cc6600 - typically 3px, exceeds requirement

Correct 5: Box-Shadow Alternative

box-shadow: 0 0 0 3px #0066cc - box-shadow not subject to outline width check

Expected Test Results:

  • Error Code: None (negative test)
  • Count: 0 errors
  • WCAG: 2.4.11 Focus Appearance (Level AAA)
  • Why: All outline widths meet or exceed 2px requirement
  • Details:
    • Correct 1: 2px outline - minimum AAA
    • Correct 2: 3px outline - exceeds AAA
    • Correct 3: 4px outline - excellent
    • Correct 4: "medium" keyword (~3px)
    • Correct 5: Box-shadow alternative (not subject to outline width check)
  • Best Practice: Use 3px or greater for optimal visibility across diverse user needs
