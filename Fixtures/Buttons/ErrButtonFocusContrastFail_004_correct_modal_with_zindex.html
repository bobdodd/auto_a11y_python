Test: Modal Dialog with Z-Index (Should Pass)

This tests the scenario where parent elements have z-index (modal overlay, sidebar), but buttons sit on containers with solid backgrounds. Since we find a solid background BEFORE reaching the z-indexed parent, we CAN verify contrast and should not issue warnings.

Confirm Action

This is a modal dialog with z-index: 1000. However, the modal itself has a solid white background, so button focus outline contrast can be verified.

Key point: Although the overlay has z-index: 1000, the buttons sit inside the modal dialog which has background: white. The background walker finds this solid white background BEFORE reaching the z-indexed overlay, so it stops and uses white for contrast calculation. This is the expected behavior for modal dialogs.
