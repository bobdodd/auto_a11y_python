WarnTabindexDefaultFocus - Correct Examples

This page contains elements with tabindex that define custom focus indicators. None should generate warnings.

Custom Focus Indicators

Correct 1: Custom Outline

Explicitly defines outline: 3px solid #0066cc with offset

Correct 2: Custom Box-Shadow

Custom box-shadow: 0 0 0 3px #cc0000 focus indicator

Correct 3: Custom Border Change

Border width changes from 1px to 3px on focus

Correct 4: Custom Alternative (no outline)

Removes outline but provides custom box-shadow alternative

Correct 5: Multiple Custom Indicators

Combines outline, box-shadow, and background color change

Expected Test Results:

  • Warning Code: None (negative test)
  • Count: 0 warnings
  • WCAG: 2.4.7 Focus Visible (Level AA)
  • Why: All elements define custom focus indicators
  • Details:
    • Correct 1: Custom outline with offset
    • Correct 2: Custom box-shadow
    • Correct 3: Custom border width change
    • Correct 4: Box-shadow alternative to outline
    • Correct 5: Multiple custom indicators combined
  • Best Practice: Always define custom focus indicators for consistency across browsers and platforms
