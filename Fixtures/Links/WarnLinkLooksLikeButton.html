WarnLinkLooksLikeButton Test Fixture

This page contains various examples of links styled to look like buttons, which should trigger the WarnLinkLooksLikeButton accessibility warning.

Primary Violations - Links That Look Like Buttons

⚠️ Accessibility Issue: These are all anchor elements (<a>) styled to look like buttons. Screen readers will announce them as links, but visual users will perceive them as buttons, creating confusion about expected behavior.

Navigation link styled as primary button:

Go to Dashboard

External link styled as secondary button:

Visit Example.com

Download link styled as danger button:

Delete Account

Page link styled as large button:

View Profile
<a href="/dashboard" class="button-link">Go to Dashboard</a> <a href="https://example.com" class="secondary-button-link">Visit Example.com</a> <a href="/download/file.pdf" class="danger-button-link">Delete Account</a>
