Color Scheme Support

This page demonstrates the correct use of the color-scheme property on the <html> element.

What is color-scheme?

<html style="color-scheme: dark light;">

The color-scheme property tells the browser which color schemes the page supports. This enables:

  • Native dark mode for form controls (inputs, selects, buttons)
  • System UI theming for scrollbars and native widgets
  • Automatic adjustments based on user preferences
  • Better performance than JavaScript-based theme switching for system elements

Why This Is An Accessibility Feature

The color-scheme property provides important accessibility benefits:

  • Respects user preferences - honors system-wide dark mode settings
  • Reduces eye strain - dark mode is easier on the eyes in low-light environments
  • Benefits users with photophobia - reduces light sensitivity issues
  • Improves readability - some users read better with dark backgrounds
  • Saves battery on OLED screens (accessibility benefit for mobile users)

Combined with CSS Variables

The color-scheme property works great with CSS variables for comprehensive theming:

<html style="color-scheme: dark light; --primary-color: #007bff;">

This combination allows both native system UI theming AND custom application theming.

Testing Dark Mode

To test this page in dark mode:

  • macOS: System Preferences → General → Appearance → Dark
  • Windows: Settings → Personalization → Colors → Choose your color → Dark
  • iOS/iPadOS: Settings → Display & Brightness → Dark
  • Android: Settings → Display → Dark theme
