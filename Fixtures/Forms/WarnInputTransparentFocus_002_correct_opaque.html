WarnInputTransparentFocus - Correct Examples

This page contains text input fields with opaque or mostly opaque focus indicators (alpha ≥ 0.5). None of these should generate warnings.

Opaque Focus Indicators

Expected Test Results:

  • Warning Code: None (negative test)
  • Count: 0 warnings
  • WCAG: 2.4.7 Focus Visible (Level AA), 1.4.11 Non-text Contrast (Level AA)
  • Why: All focus indicators are opaque enough (alpha ≥ 0.5) for good visibility
  • Opacity Levels:
    • Input 1-2: 100% opaque (ideal)
    • Input 3: 90% opaque (excellent)
    • Input 4: 70% opaque (good)
    • Textarea: 50% opaque (minimum acceptable)
  • Best Practice: Fully opaque (alpha = 1.0) is RECOMMENDED for maximum visibility
