ErrTabindexTransparentOutline - Violation Examples

This page contains elements with tabindex that have semi-transparent focus indicators (alpha < 0.5). All should generate errors.

Transparent Focus Indicators

Violation 1: Very Transparent (alpha 0.2)

outline: 3px solid rgba(0, 102, 204, 0.2) - barely visible

Violation 2: Moderately Transparent (alpha 0.4)

outline: 3px solid rgba(204, 0, 0, 0.4) - still below 0.5 threshold

Violation 3: Transparent Box-Shadow (alpha 0.3)

box-shadow with rgba(0, 102, 0, 0.3) - too transparent

Violation 4: Nearly Invisible (alpha 0.1)

outline with alpha 0.1 - effectively invisible

Violation 5: Transparent Keyword

outline: 3px solid transparent - completely invisible
