Why This Works

These buttons use the outline CSS property for focus indication, which provides:

  • Visibility at any zoom level: Outline is visible even when highly magnified
  • Clear boundary marker: Wraps the entire button edge, making focus obvious
  • Persistence: Remains visible when user looks away and returns
  • Independence from background: Works regardless of button color or size
  • High contrast: White outline on dark button, orange outline on green button
  • Offset spacing: Creates clear separation from button edge for maximum visibility

The background color may also change as a supplementary indicator, but the outline ensures WCAG 2.4.7 compliance by providing a clear, persistent focus marker.
