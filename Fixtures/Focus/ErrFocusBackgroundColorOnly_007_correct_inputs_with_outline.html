Contact Form - Accessible Version

Accessibility Implementation

These form fields use proper focus indicators that comply with WCAG 2.4.7:

Input Fields

  • 3px solid blue outline: Prominent and easily visible
  • 2px outline offset: Creates clear space around field boundary
  • Visible at high zoom: Outline remains visible even at 400% magnification
  • Persistent marker: Clearly marks focused field when user returns attention
  • Supplementary indicators: Background color and border changes provide additional cues

Submit Button

  • 3px solid orange outline: High contrast against green button
  • 3px outline offset: Creates prominent focus ring
  • Clear boundary: Wraps entire button making focus immediately obvious

Why This Approach Works

The outline property creates a visual boundary marker that:

  • Works at any magnification level (users may see only part of field but still see outline edge)
  • Provides persistent indication (visible when user returns focus after distraction)
  • Is independent of fill colors (outline wraps boundary regardless of background)
  • Meets WCAG 2.4.7 requirements for all user groups
  • Can be supplemented with other visual changes for enhanced usability
