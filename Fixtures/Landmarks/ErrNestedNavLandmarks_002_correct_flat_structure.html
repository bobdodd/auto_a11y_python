Introduction

When you need to organize links into categories within navigation, use headings and lists within a single nav element, not nested nav elements.

Best Practice: Keep all navigation landmarks at the same hierarchical level (as siblings). If you need to organize links into categories, use headings (h2, h3, etc.) and lists within a single nav element, not nested nav landmarks.

Example: Organizing links within one nav 
<nav aria-label="Site navigation">
  <h2>Main Pages</h2>
  <ul>
    <li><a href="/">Home</a></li>
    <li><a href="/about">About</a></li>
  </ul>

  <h2>Resources</h2>
  <ul>
    <li><a href="/guides">Guides</a></li>
    <li><a href="/faq">FAQ</a></li>
  </ul>
</nav>
