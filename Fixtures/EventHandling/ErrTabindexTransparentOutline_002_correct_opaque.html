ErrTabindexTransparentOutline - Correct Examples

This page contains elements with tabindex that have opaque or sufficiently visible focus indicators (alpha ≥ 0.5 or solid). None should generate errors.

Opaque Focus Indicators

Correct 1: Fully Opaque (alpha 1.0)

outline: rgba(0, 102, 204, 1.0) - completely visible

Correct 2: Minimum Acceptable (alpha 0.5)

outline: rgba(204, 0, 0, 0.5) - meets minimum threshold

Correct 3: Mostly Opaque (alpha 0.8)

box-shadow: rgba(0, 102, 0, 0.8) - highly visible

Correct 4: Solid Color

outline: #0066cc - no alpha channel, fully opaque

Correct 5: High Opacity (alpha 0.9)

box-shadow: rgba(204, 102, 0, 0.9) - excellent visibility

Expected Test Results:

  • Error Code: None (negative test)
  • Count: 0 errors
  • WCAG: 2.4.7 Focus Visible (Level AA)
  • Why: All focus indicators have sufficient opacity (alpha ≥ 0.5 or solid)
  • Details:
    • Correct 1: alpha 1.0 (fully opaque)
    • Correct 2: alpha 0.5 (minimum threshold)
    • Correct 3: alpha 0.8 (mostly opaque)
    • Correct 4: Solid color (no alpha)
    • Correct 5: alpha 0.9 (high opacity)
  • Best Practice: Prefer fully opaque colors (alpha 1.0 or solid hex/rgb) for maximum visibility
