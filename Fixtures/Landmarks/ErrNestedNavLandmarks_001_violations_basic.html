Understanding Navigation Nesting Issues

Navigation landmarks should not be nested inside other navigation landmarks. Each nav should be a distinct, top-level navigation region.

Nested navigation creates confusion about the navigation hierarchy and makes it difficult for screen reader users to understand the relationship between navigation sections.
