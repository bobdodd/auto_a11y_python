Alternative Pattern

Using role="main" on a div element is also valid and creates the main landmark.

However, using the semantic HTML5 main element is preferred when possible.

Best Practice: Every page should have one main landmark. Use the HTML5 main element or role="main" on a div. This is a WCAG 1.3.1 Level A requirement.
