About Dictaphone Format

Dictaphone generates accessibility findings from audio/video recordings of manual audits. The JSON format includes:

  • Recording ID: Unique identifier
  • Issues: List of accessibility problems found
  • Timecodes: Start/end times in the recording
  • Impact Levels: High, Medium, Low
  • WCAG Criteria: Relevant guidelines
  • Detailed Descriptions: What, why, who, and how to fix
Example JSON Structure
{
  "recording": "NED-A",
  "issues": [
    {
      "title": "Issue title",
      "impact": "high",
      "timecodes": [...],
      "wcag": [...]
    }
  ]
}
