Test Instructions

  • Run accessibility checker on this page
  • Should detect multiple instances of ErrButtonTextLowContrast
  • All buttons in the "Failing Cases" section should trigger the error
  • No buttons in the "Passing Cases" section should trigger the error
  • Test interactive states by hovering and focusing on the state-specific test buttons
  • Verify that the checker properly calculates contrast ratios for complex styling (gradients, transparency)
  • Check that large text thresholds are properly applied
