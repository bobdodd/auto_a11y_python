WarnTabindexNoBorderOutline - Warning Examples

This page contains elements with tabindex that use ONLY outline for focus indication. Screen magnifier users may miss outline if zoomed. All should generate warnings.

Outline-Only Focus Indicators

Warning 1: Basic Outline Only

outline: 3px solid - no border, box-shadow, or background change

Warning 2: Outline with Offset

Outline with offset but no border/box-shadow alternative

Warning 3: Thick Outline (5px)

Even thick outlines can be missed by screen magnifier users

Warning 4: Dotted Outline

Different outline style but still outline-only

Warning 5: Outline + Non-Indicator Props

Has outline plus cursor/z-index but no border/box-shadow/background
