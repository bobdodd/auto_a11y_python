Welcome to Our Site

This page follows the most common and recommended pattern: a single banner landmark for the site header.

This is clear, unambiguous, and easy for screen reader users to navigate.

Important Information

If you need additional header-like sections, consider using other landmarks:

  • Use complementary (aside) for promotional content
  • Use region with aria-label for important notices
  • Use section elements without roles for general grouping

Best Practice: Use exactly one banner landmark per page. Multiple banners are uncommon and should only be used when absolutely necessary, with unique labels for each.

Alternative: Multiple labeled banners (if needed)

If you must have multiple banner landmarks, structure them like this:

<header role="banner" aria-label="Main site header">
  <!-- Main site header content -->
</header>

<div role="banner" aria-label="Promotional banner">
  <!-- Promotional content -->
</div>

However, this pattern is rarely needed. Consider using complementary or region landmarks instead.
