Color Scheme in Style Tag

About This Test

This page has a <style> tag that sets color-scheme on the html element.

<style>
    html {
        color-scheme: dark light;
    }
</style>

Why This Should Pass

  • The color-scheme property is an accessibility feature
  • It enables native dark mode for form controls and system UI
  • It respects user preferences for color schemes
  • This is the recommended way to enable dark mode support

Test Expectation

Expected Result: The style tag should NOT be flagged because it only sets color-scheme on html. Actual styling is done via external CSS which is allowed.

Form Example


Form controls will automatically use dark styling when system prefers dark mode.
