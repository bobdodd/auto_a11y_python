CSS Variables Example

This page demonstrates the correct use of CSS custom properties (CSS variables) on the <html> element.

Design System Variables

The HTML element defines these CSS variables:

  • --primary-color: #007bff - Used for primary actions
  • --secondary-color: #6c757d - Used for secondary elements
  • --font-size-base: 16px - Base font size for the entire document
  • --spacing-unit: 8px - Base spacing unit for consistent spacing

Why This Is Correct

CSS custom properties (variables) on the <html> element are:

  • Standard practice for design systems and theming
  • User-customizable - can be overridden with user stylesheets
  • Maintainable - centralized values that cascade to all elements
  • Dynamic - can be changed via JavaScript for theme switching
  • Not hard-coded styles - they define reusable values, not direct styling

Accessibility Benefits

CSS variables provide accessibility benefits:

  • Users can override variables with their own stylesheets
  • Browser extensions can modify variables for high contrast modes
  • JavaScript can adjust variables based on user preferences
  • They enable consistent spacing and sizing (important for WCAG 1.4.12)
