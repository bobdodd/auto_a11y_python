WarnInputDefaultFocus - Warnings

This page contains text input fields that rely on default browser focus styles (no custom :focus styles defined). While browser defaults are functional, they vary across browsers (Chrome: blue outline, Firefox: blue dotted, Safari: blue ring) and may not match your site's design. Custom focus styles are RECOMMENDED for consistency. All 4 inputs should generate warnings.

Default Browser Focus Styles

Expected Test Results:

  • Warning Code: WarnInputDefaultFocus
  • Count: 4 warnings
  • WCAG: 2.4.7 Focus Visible (Level AA) - technically passes
  • Why: Default browser focus styles vary across browsers and may not match your site's design or branding
  • Browser Variations:
    • Chrome/Edge: Blue outline (rgb(0, 103, 244)) ~1px
    • Firefox: Blue dotted outline (rgb(94, 158, 214))
    • Safari: Blue ring (rgb(77, 144, 254))
  • Recommendation: Define custom focus styles for visual consistency
  • Note: This is a design/consistency concern, not a WCAG violation
