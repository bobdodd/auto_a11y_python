Duplicate Complementary Landmark Labels - Violations

Main Article: Understanding Complementary Landmarks

Complementary landmarks (aside elements with role="complementary") provide supporting content. When multiple exist, each must have a unique accessible name.

When multiple complementary landmarks exist with the same label, screen reader users have no way to distinguish between them. Each must have a unique, descriptive accessible name.
