Testing Accessible Fonts

Arial (Default)

This text uses Arial, a clean, sans-serif font with good character distinction. It's widely recommended for accessibility as it's easy to read and available on most systems.

Verdana

This text uses Verdana, designed specifically for screen readability. It has generous spacing between characters and is excellent for users with dyslexia or low vision.

Georgia

This text uses Georgia, a serif font that maintains good readability at various sizes. While serif fonts are less common for body text, Georgia is considered accessible due to its clear letterforms.

Tahoma

This text uses Tahoma, a sans-serif font with tight letter spacing but still maintains good legibility. It's appropriate for accessibility when used at appropriate sizes.
