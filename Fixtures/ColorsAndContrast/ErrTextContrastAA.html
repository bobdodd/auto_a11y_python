Text Contrast AA Test Fixture

Purpose: This fixture contains multiple text elements that fail WCAG 2.1 Level AA contrast requirements (4.5:1 for normal text) to test the ErrTextContrastAA error detection.

Test Case 1: Light Gray on White (~2.9:1)

This text uses #9d9d9d on #ffffff background with 16px font size. The contrast ratio is approximately 2.9:1, which fails WCAG AA requirements. This should trigger an ErrTextContrastAA error.

Test Case 2: Gray on Light Gray (~2.1:1)

This paragraph demonstrates #888888 text on #e8e8e8 background at 14px. The contrast ratio is around 2.1:1, significantly below the required 4.5:1 threshold.

Test Case 3: Light Blue on White (~3.2:1)

Here we have #7fb3d3 colored text on white background (#ffffff) at 18px font size. This creates approximately 3.2:1 contrast ratio, failing AA compliance.

Test Case 4: Dark Gray on Medium Gray (~2.8:1)

This content uses #666666 foreground color on #cccccc background with 16px font. The resulting contrast of ~2.8:1 does not meet accessibility standards.

Test Case 5: Yellow on Light Yellow (~1.8:1)

This text demonstrates #d4aa00 on #fffacd background at 20px font size. With only 1.8:1 contrast ratio, this is particularly problematic for users with visual impairments.

Test Case 6: Purple on Light Purple (~3.8:1)

Using #8b5a8c text on #f0e6f7 background with 15px font creates approximately 3.8:1 contrast. While better than previous examples, it still falls short of the 4.5:1 requirement.

Test Case 7: Orange on Peach (~2.5:1)

This demonstrates #cc7a00 text on #ffe5cc background at 17px. The 2.5:1 contrast ratio makes this text difficult to read for many users.

Test Case 8: Green on Light Green (~3.1:1)

Here's #5d8a5d colored text on #e8f5e8 background with 19px font size. The contrast ratio of approximately 3.1:1 fails to meet WCAG AA standards.

Test Case 9: Red on Pink (~4.2:1 - Close Failure)

This text uses #b33939 on #ffeaea background at 16px. With 4.2:1 contrast, it's close to passing but still fails the 4.5:1 AA requirement by a narrow margin.

Test Case 10: Very Light Gray on White (~1.1:1)

This extremely low contrast example uses #f0f0f0 on #ffffff with 14px font. At only 1.1:1 contrast ratio, this text is nearly invisible to many users.

Test Case 11: Small Text Edge Case

This smaller text (12px) uses #767676 on white background. While large text might pass at lower contrast, normal text at this size with ~4.4:1 contrast fails AA requirements.
