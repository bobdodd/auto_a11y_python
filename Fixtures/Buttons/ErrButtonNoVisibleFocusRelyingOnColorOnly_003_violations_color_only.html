Test: outline:none with Color Change Only

These buttons remove the focus outline and only change color, which fails WCAG 1.4.1 Use of Color. Users with color blindness or monochrome displays cannot perceive focus state.

Background Color Change Only

This button removes outline and only changes background color on focus.

Border Color Change Only

This button removes outline and only changes border color on focus.

Text Color Change Only

This button removes outline and only changes text/background colors on focus.
