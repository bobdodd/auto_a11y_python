ErrInputOutlineWidthInsufficient - Violations

This page contains text input fields with outline focus indicators less than 2px wide. While 1px outlines meet WCAG AA (2.4.7), WCAG AAA (2.4.11 Focus Appearance) recommends a minimum 2px outline width for better visibility. All 4 inputs should be flagged.

Thin Outline Focus Indicators

Expected Test Results:

  • Error Code: ErrInputOutlineWidthInsufficient
  • Count: 4 violations
  • WCAG: 2.4.7 Focus Visible (Level AA), 2.4.11 Focus Appearance (Level AAA)
  • Why: WCAG 2.4.11 Focus Appearance recommends minimum 2px outline width for optimal visibility
  • Note: This is a Level AAA criterion, so it's a recommendation rather than a strict requirement
