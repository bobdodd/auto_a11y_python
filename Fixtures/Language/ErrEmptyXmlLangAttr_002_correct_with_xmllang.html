Technical Documentation

This XHTML document correctly specifies both lang="en" and xml:lang="en" on the html element. This ensures compatibility with both HTML5 parsers (which use lang) and XML parsers (which use xml:lang).

XHTML Best Practices

In XHTML documents, always include both lang and xml:lang attributes with identical values. This provides maximum compatibility and ensures assistive technologies can properly identify the document language regardless of the parsing context.
