WarnTabindexDefaultFocus - Warning Examples

This page contains elements with tabindex that rely on browser default focus indicators without custom styling. All should generate warnings.

Browser Default Focus Indicators

Warning 1: No Focus Styles Defined

No :focus rule at all - relies entirely on browser default

Warning 2: Empty Focus Rule

:focus rule exists but has no properties - browser default used

Warning 3: Non-Indicator Properties Only

Focus rule changes cursor and user-select but no visual indicator

Warning 4: Comment Only

Focus rule contains only comments - no actual styles

Warning 5: Unrelated Properties

Focus rule changes z-index and position but no visual indicator

Expected Test Results:

  • Warning Code: WarnTabindexDefaultFocus
  • Count: 5 warnings detected
  • WCAG: 2.4.7 Focus Visible (Level AA)
  • Why: Relying on browser defaults can be inconsistent across browsers/platforms
  • Details:
    • Warning 1: No :focus rule defined
    • Warning 2: Empty focus rule (no properties)
    • Warning 3: Only non-visual properties changed
    • Warning 4: Only comments in focus rule
    • Warning 5: Unrelated properties (z-index, position)
  • Impact: Browser defaults vary - may not be consistent or visible enough
  • Recommendation: Define explicit focus indicators (outline, box-shadow) for consistency
  • Note: This is a WARNING, not an error - browser defaults are functional but not ideal
