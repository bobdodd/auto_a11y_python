CSS Variables in Style Tag

About This Test

This page has a <style> tag that targets the html element with CSS custom properties.

<style>
    html {
        --primary-color: #007bff;
        --secondary-color: #6c757d;
        --font-size-base: 16px;
        --spacing-unit: 8px;
    }
</style>

Why This Should Pass

  • The style tag contains ONLY CSS custom properties targeting html
  • No hard-coded color or font properties are applied to html
  • CSS variables are a legitimate pattern for design systems
  • Users can still override these variables with their own stylesheets

Test Expectation

Expected Result: The style tag should NOT be flagged as ErrStyleTagColorFont because it only sets CSS variables on the html element. Actual styling is done via external CSS which is allowed.
