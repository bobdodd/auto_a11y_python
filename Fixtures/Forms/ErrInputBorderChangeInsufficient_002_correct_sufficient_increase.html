ErrInputBorderChangeInsufficient - Correct Examples

This page contains text input fields with sufficient border thickness increases (≥1px) or alternative focus indicators. None of these should be flagged.

Sufficient Border Changes and Alternatives

Expected Test Results:

  • Error Code: None (negative test)
  • Count: 0 violations
  • WCAG: 2.4.7 Focus Visible (Level AA)
  • Why: All border increases are ≥1px, or alternative focus indicators (outline/box-shadow) are used
