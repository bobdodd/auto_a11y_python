Reduced Motion Support Test Fixture

This page contains various animations that DO NOT respect the prefers-reduced-motion setting. All animations should be disabled or significantly reduced when the user has enabled reduced motion preferences.

Test Instructions:

To test this fixture:

  1. Enable "Reduce motion" in your OS accessibility settings
  2. Refresh this page
  3. All animations should stop or be significantly reduced
  4. Currently, they will continue running (demonstrating the error)

1. Spinning Logo Animation

Continuously rotating element that can cause dizziness

2. Bouncing Ball

Vertical bouncing motion that can trigger vestibular issues

3. Sliding Banner

This banner slides back and forth continuously

Horizontal sliding motion that can be disorienting

4. Pulsing Button

Scale and opacity changes that can be distracting for users with ADHD

5. Shaking Alert

⚠️ Warning: This alert is shaking!

Rapid shaking motion that can trigger seizures in sensitive users

6. 3D Rotating Card

Rotating Card

3D rotation that can cause motion sickness

7. Zooming Image

Scale animation that can be overwhelming for motion-sensitive users

8. Parallax Section

Parallax Effect

Subtle parallax motion that should be disabled for reduced motion users

9. Flickering Text

This text flickers rapidly

Opacity changes that can trigger photosensitive seizures

10. Interactive Hover/Click Animations

Interactive animations that should also respect motion preferences

Expected Behavior with Reduced Motion:

  • All continuous animations should stop
  • Transform animations should be replaced with simple opacity or color changes
  • Interactive animations should be instant or very brief
  • Parallax and scrolling effects should be disabled
  • Flickering or rapid changes should be eliminated

Accessibility Impact:

  • Vestibular Disorders: Motion can cause nausea, dizziness, and disorientation
  • Photosensitive Epilepsy: Flickering and rapid changes can trigger seizures
  • ADHD: Animations can be extremely distracting and prevent focus
  • Cognitive Disabilities: Motion can interfere with content comprehension
