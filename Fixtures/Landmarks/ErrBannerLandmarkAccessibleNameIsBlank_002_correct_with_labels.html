This page demonstrates banner landmarks with proper accessible names. When multiple banner landmarks exist on a page, each must have a unique, descriptive accessible name to help screen reader users distinguish between them.

Note: While having multiple banner landmarks is uncommon, if they exist (e.g., main header, promotional banner, account banner), they must be properly labeled.
