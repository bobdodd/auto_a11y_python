ErrTabindexColorChangeOnly - Correct Examples

This page contains elements with tabindex that have structural focus indicators (outline, box-shadow, or border width change). None should generate errors.

Structural Focus Indicators

Correct 1: Outline + Color Change

Adds outline (structural) AND changes background color - compliant

Correct 2: Box-Shadow + Color

Box-shadow provides structural indicator, color enhances it

Correct 3: Border Width Change

Border width changes from 1px to 3px - structural change

Correct 4: Outline Only

Outline provides structural indicator without needing color change

Correct 5: Box-Shadow Only

Box-shadow alone provides sufficient structural indicator

Expected Test Results:

  • Error Code: None (negative test)
  • Count: 0 errors
  • WCAG: 1.4.1 Use of Color (Level A)
  • Why: All have structural indicators, not relying on color alone
  • Details:
    • Correct 1: Outline + color (structural present)
    • Correct 2: Box-shadow + color (structural present)
    • Correct 3: Border width change (structural)
    • Correct 4: Outline only (no color needed)
    • Correct 5: Box-shadow only (no color needed)
  • Best Practice: Always provide outline or box-shadow; color changes can enhance but not replace structural indicators
