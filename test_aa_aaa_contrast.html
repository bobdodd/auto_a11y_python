AA vs AAA Contrast Testing

This page tests different contrast scenarios for WCAG AA and AAA compliance.

Normal Text Tests (14px)
4.69:1 ratio - Should PASS AA (needs 4.5:1), Should FAIL AAA (needs 7:1)
7.5:1 ratio - Should PASS both AA and AAA
3.5:1 ratio - Should FAIL AA (needs 4.5:1)
Large Text Tests (24px)
3.5:1 ratio - Should PASS AA (needs 3:1), Should FAIL AAA (needs 4.5:1)
4.69:1 ratio - Should PASS both AA (needs 3:1) and AAA (needs 4.5:1)
2.5:1 ratio - Should FAIL AA (needs 3:1)
Expected Results

When testing with WCAG Level AA:

  • Normal text 4.69:1 - PASS (uses ErrTextContrastAA if failed)
  • Normal text 3.5:1 - FAIL with ErrTextContrastAA
  • Large text 3.5:1 - PASS (uses ErrLargeTextContrastAA if failed)
  • Large text 2.5:1 - FAIL with ErrLargeTextContrastAA

When testing with WCAG Level AAA:

  • Normal text 4.69:1 - FAIL with ErrTextContrastAAA
  • Normal text 7.5:1 - PASS
  • Large text 3.5:1 - FAIL with ErrLargeTextContrastAAA
  • Large text 4.69:1 - PASS
