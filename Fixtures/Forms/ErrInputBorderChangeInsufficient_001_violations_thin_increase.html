ErrInputBorderChangeInsufficient - Violations

This page contains text input fields where the border thickness increases by less than 1px on focus. Border thickening is acceptable as a focus indicator, but the change must be manifest (≥1px) so users can clearly notice the difference. All 4 inputs should be flagged.

Insufficient Border Thickness Increases

Expected Test Results:

  • Error Code: ErrInputBorderChangeInsufficient
  • Count: 4 violations
  • WCAG: 2.4.7 Focus Visible (Level AA)
  • Why: Border thickness increase must be ≥1px to be manifest (clearly noticeable) to users
  • Note: Input fields often have borders to identify the input area, so border thickening is acceptable, but only if the change is noticeable
