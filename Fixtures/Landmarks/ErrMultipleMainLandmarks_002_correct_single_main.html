Critical Rule: Use exactly ONE main landmark per page. The main should contain all the primary, unique content for that page. Use semantic HTML elements (article, section) and headings (h2-h6) to organize different content areas within the single main landmark.

How to structure complex pages with one main 
<main>
  <section>
    <h2>Featured Articles</h2>
    <article>...</article>
    <article>...</article>
  </section>

  <section>
    <h2>Products</h2>
    <!-- Product grid -->
  </section>

  <section>
    <h2>Dashboard</h2>
    <!-- Dashboard widgets -->
  </section>
</main>

Use sections, articles, and headings to organize - NOT multiple main elements!
