About Auto A11y

Auto A11y is an automated web accessibility testing tool designed to help developers and testers identify and fix accessibility issues in web applications.

Features
  • Automated WCAG 2.2 compliance testing
  • Lived experience testing with real users
  • Manual accessibility audits and recordings
  • Comprehensive reporting and issue tracking
  • Project and test user management
WCAG Data Attribution

This application uses official WCAG (Web Content Accessibility Guidelines) data from the World Wide Web Consortium (W3C).

Source

The WCAG Success Criteria data used in this application is sourced from:

  • Web Content Accessibility Guidelines (WCAG) 2.2
    https://www.w3.org/TR/WCAG22/

    https://www.w3.org/WAI/WCAG22/Techniques/

Conformance Levels

Auto A11y tests against all three WCAG conformance levels:

  • Level A: Minimum level of conformance
  • Level AA: Standard level (most common target)
  • Level AAA: Highest level of conformance
Testing Approaches

Auto A11y supports multiple testing approaches:

  • Automated Testing: Using axe-core engine
  • Manual Audits: Human expert evaluation
  • Lived Experience: Testing by people with disabilities
