Test Instructions
Expected Results:
  • All text in "FAILING Cases" sections should trigger ErrInsufficientContrast
  • Text in "PASSING Cases" section should NOT trigger this error
  • The error should identify the specific contrast ratios that fail requirements
  • Both normal text (< 4.5:1) and large text (< 3:1) failures should be detected
  • Various color combinations and UI elements should be tested
Edge Cases Covered:
  • Barely failing contrast ratios (just below thresholds)
  • Different color combinations (not just gray-scale)
  • Form elements and interactive content
  • Large text vs normal text thresholds
  • Various background colors
